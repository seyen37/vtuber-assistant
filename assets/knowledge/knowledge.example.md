# 知識庫（客服範例）— 換成你自己的內容

> 用法：把這個檔換成你的 FAQ / 產品說明 / 客服話術。以「空行」分段，每段是一條可被檢索的知識。
> 開啟方式：⚙ 設定 → 知識庫客服 → 勾「啟用」。

## 營業時間
我們的客服服務時間為週一至週五 09:00–18:00（國定假日休息）。非營業時間可留言，將於下一個工作日回覆。

## 退換貨政策
商品到貨後 7 天內，於商品完整、未使用、包裝齊全的情況下可申請退換貨。退貨運費由買方負擔；若為瑕疵品則由本公司負擔。

## 運送與到貨時間
本島宅配一般於下單後 2–3 個工作日送達；外島約 3–5 個工作日。下單後可於「訂單查詢」頁面追蹤物流狀態。

## 付款方式
支援信用卡、ATM 轉帳、超商代碼繳費與貨到付款。信用卡支援分期（依銀行方案）。
